## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Database Config
MongoDb is used by default. Before running the application database config need to check in database.config.ts file

## Running the app

```bash
# development
$ npm run start
```
